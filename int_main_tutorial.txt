В ANSI C 89 разница между следующими функциями:
int main()
int main(void)
int main(int argc, char* argv[])
это:

int main():
функция, которая ожидает неизвестное количество аргументов неизвестных типов.
Возвращает целое число, представляющее состояние прикладного программного обеспечения.

int main(void):
функция, которая не ожидает аргументов.
Возвращает целое число, представляющее состояние прикладного программного обеспечения.

int main(int argc, char * argv[]):
функция, которая ожидает количество аргументов argc и argv[] аргументы.
Возвращает целое число, представляющее состояние прикладного программного обеспечения.

О том, как использовать каждую из функций

int main(void):
нужно использовать эту функцию, когда в программе НЕ нужны
начальные параметры для запуска / загрузки (параметры, полученные от операционной системы - из самой программы).

int main(int argc, char * argv[]):
нужно использовать эту функцию, когда в программе нужны
начальные параметры для загрузки (параметры, полученные из операционной системы - из самой программы).
